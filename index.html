I'm a full stack developer who enjoys turning complex problems into clean, maintainable software. My work spans the whole stack — building backends in Go and Node.js / TypeScript, modeling data with PostgreSQL, MongoDB, Redis and Prisma, and crafting modern interfaces with React.

I care about the fundamentals: Hexagonal / Clean Architecture, SOLID principles, validation, security and observability. I've built everything from Go microservices communicating over gRPC with event-driven RabbitMQ flows, to a production-grade observability platform with Redis Streams and REST APIs with auth, rate limiting and Swagger docs.

I'm always learning — currently deepening my knowledge of distributed systems, testing and DevOps practices to ship software that scales with confidence.

02 — Skills

What I Do

Backend Engineering

Robust REST APIs with Go, Node.js, TypeScript, Express and Gin. JWT auth, rate limiting, validation with Zod, structured logging and Swagger documentation.

Frontend Development

Responsive, accessible UIs with React, TypeScript and Vite. State management with Zustand and React Query, plus clean component design.

Data & Databases

PostgreSQL, SQLite, MongoDB and TimescaleDB. Caching and streaming with Redis, modeling with Prisma, Sequelize and Mongoose.

Architecture

Clean & Hexagonal Architecture, layered design, distributed systems and microservices. Separation of concerns, testability and scalability by design.

Security

Authentication & authorization, input sanitization, security headers, rate limiting, TLS and idempotent webhooks — applied OWASP practices throughout.

DevOps & Tooling

Docker & Docker Compose, Kubernetes, Prometheus, Git workflows, monorepos and CI/CD pipelines.
